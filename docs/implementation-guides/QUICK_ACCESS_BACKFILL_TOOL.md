1. **Log in** to your admin account
2. Look at the **left sidebar navigation**

## What It Looks Like

When you access the backfill tool, you'll see:

1. **Header** - Purple gradient with "Image Signals Backfill Tool" title
2. **Info Box** - Blue box explaining what the tool does
3. **Stats Dashboard** (when running):
   - Total payments
   - Processed count
   - Successful count
   - Skipped count
   - Failed count
   - Progress bar

4. **Start Backfill Button** - Large purple button
5. **Activity Log** - Black terminal-style log showing real-time processing

---

## Quick Start Steps

1. ✅ Navigate to the tool (see above)
2. ✅ Read the info box to understand what it does
3. ✅ Click **"Start Backfill"**
4. ✅ Watch the progress
5. ✅ Wait for completion message
6. ✅ Done!

**Time Required:** 5-15 minutes depending on database size

**Safe to Run:** Yes! Can be interrupted and restarted

## Troubleshooting

### "I don't see Admin Tools in the sidebar"

**Solution:** You might be logged in as an occupant or committee member. The tool is only accessible to **Admin** or **Super Admin** users.

1. Log out
2. Log in with admin credentials
3. Check the sidebar again

---

### "I see Admin Tools but not the backfill tool"

**Possible Causes:**
1. Page didn't load fully - try refreshing
2. JavaScript error - check browser console
3. Old cached version - clear browser cache

**Solution:**
1. Refresh the page (F5 or Ctrl+R)
2. Clear browser cache
3. Try a different browser

---

### "The tool is stuck or not responding"

**Solution:**
1. Check your internet connection
2. Look at the activity log for errors
3. If stuck, refresh the page and restart
4. Progress is saved - already-processed payments will be skipped
